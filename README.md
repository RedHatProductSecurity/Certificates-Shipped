# Certificates-Shipped
A list of certificates shipped by various vendors and Open Source Projects

The goal here is to list all certificates, keys, etc. including but not limited 
to:

* .asc
* .ber
* .cer
* .cert
* .csr
* .crt
* .der
* .gpg
* .jks
* .key
* .nss
* .p12
* .p7s
* .pem 
* .pfx
* .pgp
* .xml
